## Verified (checked 2026-07-02)

- **Scientific name / taxonomy:** *Ambystoma mexicanum*, family Ambystomatidae (mole salamanders), order Urodela/Caudata, class Amphibia — confirmed via IUCN Red List, AmphibiaWeb, and Animal Diversity Web. It is an amphibian (not a fish/reptile) — correct.
- **IUCN Red List status:** **Critically Endangered** — confirmed (listed CR since 2019/2006; wild population estimated at only tens to ~1,000 adults, restricted to remnant canals of Lake Xochimilco). Book text correct. Sources: IUCN Red List; National Geographic; Wikipedia (citing IUCN).
- Regeneration, courtship, and cultural specifics remain flagged above; not individually re-verified here.
