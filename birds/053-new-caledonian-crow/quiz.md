# Book 053 — The New Caledonian Crow: Short Quiz

1. The New Caledonian crow's scientific name is:
   A. *Corvus moneduloides*
   B. *Corvus corax*
   C. *Corvus monedula*
   D. *Corvus frugilegus*

2. Which material does the crow cut into naturally barbed tools?
   A. Grass stems
   B. Bark strips
   C. Pandanus leaf edges
   D. Feather shafts

3. "Meta-tool use" in this crow refers to:
   A. Using one tool to obtain another tool
   B. Using two tools at once
   C. Using a tool with the foot
   D. Storing tools for later

4. The Aesop's fable task tests whether the crow will:
   A. Bend wire into a hook
   B. Drop objects into water to raise a floating reward
   C. Fish for termites
   D. Crack nuts with stones

5. The captive crow "Betty" became famous for:
   A. Solving a maze
   B. Mimicking human speech
   C. Bending wire into a hook to retrieve food
   D. Counting objects

6. A key reason a small crow brain can support complex cognition is:
   A. It has no need for a cortex
   B. It is larger than a chimpanzee's
   C. It uses echolocation
   D. Very high neuron density in the pallium

7. The New Caledonian crow is found naturally:
   A. Across all of Australia
   B. Throughout Southeast Asia
   C. Only in New Caledonia and immediate neighbors
   D. Worldwide in cities

8. Regional differences in Pandanus tool shape have been proposed as possible evidence of:
   A. Genetic mutation
   B. Cultural transmission
   C. Random damage
   D. Seasonal migration

9. Its IUCN Red List status is:
   A. Critically Endangered
   B. Endangered
   C. Vulnerable
   D. Least Concern

10. The comparison with the chimpanzee is scientifically important because it illustrates:
    A. Common recent ancestry
    B. Convergent evolution of tool-making
    C. Identical brain structure
    D. That crows are primates

## Answer Key

1. A
2. C
3. A
4. B
5. C
6. D
7. C
8. B
9. D
10. B
