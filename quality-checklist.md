# Quality Checklist

Run every book through this gate before marking it complete in the tracker. A book passes only when every item is satisfied.

## Completeness

- [ ] All five files exist: `book.md`, `summary.md`, `glossary.md`, `quiz.md`, `references.md`.
- [ ] `book.md` contains all 20 numbered sections in order, with the correct title line.
- [ ] `book.md` is 2,500–5,000 words with no padding.
- [ ] `summary.md` is 300–500 words and stands alone.
- [ ] `glossary.md` has 10–20 terms with clear definitions.
- [ ] `quiz.md` has 10 multiple-choice questions plus an answer key.
- [ ] `references.md` lists credible sources with uncertain ones flagged.

## Scientific accuracy

- [ ] Common name, scientific name, and taxonomic group are correct.
- [ ] Behavioral claims reflect current understanding, not folklore.
- [ ] Only senses and feeding/social modes relevant to the species are included.
- [ ] Cognitive claims are proportionate (no overclaiming intelligence or emotion).
- [ ] Conservation status is stated carefully or flagged for verification.

## Sourcing

- [ ] No invented citations anywhere.
- [ ] Uncertain sources carry the flag "Further reading to verify before final publication."
- [ ] References are split into well-established vs. to-verify where useful.

## Myths section

- [ ] At least 5 myths, each with a calm scientific correction.
- [ ] Tone corrects misconceptions without mocking readers.

## Writing quality

- [ ] Species-specific throughout — no generic paragraphs reusable for any animal.
- [ ] No lines recycled verbatim from other books in the collection.
- [ ] No fake emotional or motivational language.
- [ ] No forced moral lessons; ethology lessons are about behavior/ecology/evolution.
- [ ] Anthropomorphism kept minimal and clearly framed when used.
- [ ] Reads as compact public science education, not a children's encyclopedia.

## Structure and handoff

- [ ] Folder name matches the master list slug and number exactly.
- [ ] Filenames are exactly `book.md`, `summary.md`, `glossary.md`, `quiz.md`, `references.md`.
- [ ] Section-17 glossary matches `glossary.md`; section-19 quiz matches `quiz.md`.

## Sign-off

- [ ] Book added/updated in `batch-progress-tracker.md`.
- [ ] Weak sections and citations-to-verify recorded in the batch note.
